Opening for Software Developer

A Senior Software Developer skilled at developing cutting edge web applications using Microsoft .NET programming. The role requires the design, development and implementation of applications which provide solutions to business problems. A Senior Software Developer will be expected to undertake such duties as consulting with clients about their business requirements, building applications to client specifications, customizing current application in line with changing business needs, reporting specifics of an application, evaluating the performance of an application, testing and implementing all software solutions, and meeting face-to- face with clients.

Key Duties/Responsibilities
Design and develop applications in agile methodology. Ready to adapt the role of mentor for junior developers in all aspects Strive to continually improve technical and developmental skills. Complete the unit testing

Candidate Profile
1.5 to 2 yrs of experience with ASP.NET, C#, ASP MVC 1 to 2 yrs of experience with SQL SERVER, DB Design, Writing Stored Procedures Experience working with AJAX, DHTML, HTML5, JQuery, Responsive design using bootstrap Experience with web services Experience with third party integration such as payment gateway is added advantage
