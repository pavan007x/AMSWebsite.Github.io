Features of AMS ERP

  • Flexible and Agile System that can adapt changes quickly
  • No Redundancy of data as everything is stored in centralized SQL Server 2008 database on the server
  • Simple and flexible interfaces and processes to perform key activities
  • High availability and real time data updating using web service calls and windows services
  • Access of information to students, parents and staff related to educational institute application, fee, attendance and marks on the web, using devices such as PC, laptop, smart phones and tablets
  • Notification delivery through SMS and emails for some critical activities
  • Audit logging/Alerts for various activities performed on the system
  • On demand reports MIS reports as well as daily reports delivery though emails
  • New technology and framework using ASP.NET, .NET4.0 ,C#, AJAX
